 Airplane Mode App

Airplane Mode is a simple practice application built while learning the **Frappe Framework**.  
This project helps understand core concepts like **custom DocTypes, relationships, naming rules, and workspace configuration**.



 Project Purpose

The main goal of this app is to:
- Learn Frappe Framework basics
- Practice creating custom DocTypes
- Understand linking and relationships
- Configure workspaces and forms
- Gain hands-on experience with app structure



Tech Stack

- **Framework:** Frappe
- **Language:** Python
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MariaDB (default with Frappe)



Features

- Custom DocTypes
- Naming rules implementation
- Linked DocTypes (relationships)
- Workspace customization
- Beginner-friendly structure


 App Structure

- `airplane_mode/` – Main app folder  
- `doctype/` – Custom DocTypes  
- `workspace/` – Workspace configurations  
- `hooks.py` – App hooks and settings
